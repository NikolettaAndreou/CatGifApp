🐱 Cat GIF App

📱 Description

This is a simple Android app built with **Jetpack Compose** that displays cat images and GIFs using **The Cat API**.

⚙️ Features

* View a list of cat GIFs and images
* Choose between:

  * GIFs
  * Images
  * Both
* Infinite scrolling (loads more items when you scroll)


🛠️ Technologies

* Kotlin
* Jetpack Compose
* Retrofit
* Coil

🧠 How it Works

The app makes API requests to The Cat API and displays the results in a list.

* The selected filter (GIFs / Images / Both) changes the API request
* Pagination is implemented using a page counter
* When the user scrolls near the bottom, more data is loaded

 🚀 How to Run

1. Clone the project
2. Open it in Android Studio
3. Run it on an emulator or device

 🔑 API

The app uses:
https://thecatapi.com/

✅ Summary

The app successfully:

* Supports filtering and pagination
